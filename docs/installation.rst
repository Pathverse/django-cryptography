Installation
============

Requirements
------------

* Python_ (3.10, 3.11, 3.12, 3.13, 3.14)
* Cryptography_ (2.0+)
* Django_ (4.2, 5.2, 6.0)

.. code-block:: console

   uv add django-cryptography

.. _Cryptography: https://cryptography.io/
.. _Django: https://www.djangoproject.com/
.. _Python: https://www.python.org/
